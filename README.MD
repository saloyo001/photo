# My Photography Website

## About the Project

This is my first photography website created during my second week of learning web development. The website is built using **HTML** and **CSS** and displays some of my favourite photographs.

## Features

* Home page
* Gallery page
* Navigation menu
* Basic CSS styling

## Technologies Used

* HTML
* CSS
* VSCODE

## How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in your web browser.
